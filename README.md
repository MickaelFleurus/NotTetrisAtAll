# Too Many Cubes

**Too Many Cubes** is a Tetris-style game made with **Unity 6**.

This is the first game I’ve created entirely on my own, and here is the result.

It’s essentially my own take on a game I love. There’s nothing particularly groundbreaking about it — this project was mainly a way for me to learn, practice, and build something complete from start to finish.

If you decide to give it a try, thank you.

The game supports controllers.

---

## Project info

Made with Unity 6.3 (editor version 6000.3.11f1). Should be possible to copy the source locally to build it (and modify it) on your own. 

## Controls

### Keyboard

**Move:** A / D or Arrow Keys
**Drop:** Spacebar
**Hold:** Left Ctrl
**Rotate Clockwise:** S
**Rotate Counterclockwise:** W
**Soft Drop / Speed Up Fall:** X

### Controller

**Move:** Left Stick / D-Pad
**Drop:** South Button
**Hold:** East Button
**Rotate Clockwise:** Right Shoulder
**Rotate Counterclockwise:** Left Shoulder
**Soft Drop / Speed Up Fall:** Down
